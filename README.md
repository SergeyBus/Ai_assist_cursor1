# ИИ ассистент-диетолог

## Описание

**ИИ ассистент-диетолог** — проект для автоматизированного расчета ежедневной потребности в калориях и формирования индивидуального меню с использованием искусственного интеллекта.

**Ассистент реализован в виде чат-бота Telegram:**  
[@caloriescook_bot](https://t.me/caloriescook_bot)

[![Telegram Bot](https://github.com/SergeyBus/Ai_assist_cursor1/blob/main/Screenshot_4.png?raw=true)](https://t.me/caloriescook_bot)

## Функционал

- **Расчет нормы калорий**:
  - Используется формула Миффлина-Сан Жеора для определения базовой потребности.
  - Три цели:
    - Набор веса — к норме добавляется 10%
    - Поддержание веса — расчет по норме
    - Похудение — от нормы отнимается 10%
- **Формирование недельного меню**:
  - На основе расчета пользователь получает сбалансированный рацион на 7 дней.
  - Подбор продуктов осуществляется с помощью ИИ (интеграция на базе GigaChat).
- **Будущие возможности**:
  - Заказ ингредиентов через онлайн-магазин (интеграция с сервисом доставки).

## Технологии и ресурсы

- Блок-схема создана на досках Яндекс
  [![Блок-схема](https://github.com/SergeyBus/Ai_assist_cursor1/blob/main/%D0%91%D0%BB%D0%BE%D0%BA-%D1%81%D1%85%D0%B5%D0%BC%D0%B0.pdf)]
- Основной алгоритм реализован на платформе **salebot**.
- Для анализа и диетических рекомендаций используется ИИ-модель на **GigaChat**.

## Пример запроса

- Пользователь указывает цель (похудение, поддержание, набор), рост, вес, возраст, уровень активности.
- Ассистент рассчитывает норму калорий и предлагает адаптированное меню.
